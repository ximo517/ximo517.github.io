Tomcat解压版安装教程

发表于 2020-02-17

1.Tomcat下载地址：http://tomcat.apache.org/
选择适合自己的版本，然后下载
2.安装
将下载好的压缩包进行解压，然后放在想要安装的目录，如“D:\Program Files\Apache\apache-tomcat”
3.环境变量配置
此电脑 –> 右键 –> “属性” –>点击”高级系统设置”

–> 点击”环境变量”

在系统变量部分新建一个变量名：CATALINA_HOME， 变量值：D:\Program Files\Apache\apache-tomcat

之后再找到系统变量中的Path

点击“编辑”，然后点击“新建”，填入“%CATALINA_HOME%\bin”

然后依次点击“确定”，关闭窗口
4.验证Tomcat环境变量是否配置成功
使用win+R打开“运行”，输入cmd,回车

输入 startup.bat 后再回车

之后就会出现Tomcat的启动窗口，如果没有报错或者一闪而过，那么说明启动成功了，让黑窗口保持运行，如果关掉窗口，那么，tomcat也就关掉了

如果窗口一闪而过，则启动失败，通常有两种解决方法
a.重启电脑后再次按照以上步骤
b.修改tomcat的端口

待问题解决后打开浏览器，在地址栏输入“http://localhost:8080 ”，回车，即可看到启动成功界面

JDK1.8安装教程

发表于 2020-02-17 更新于 2020-02-20

JDK的下载与安装

  • jdk8 x64下载地址
    链接
    提取码: 25th
  • jdk的安装
    下载完成后双击jdk-8u161-windows-x64.exe，点击“下一步”

    点击”更改”，选择要安装的目录

    更改完成后点击“下一步”

    正在安装
  • 安装完成后选择jre的安装目录
  • 选择完成后，点击“下一步”
  • 正在安装
  • 安装完成，点击“关闭”

    JDK环境变量配置

  • 新建JAVA_HOME环境变量,JAVA_HOME=C:\Java\jdk1.8.0_144
  • 修改path环境变量,在path变量尾部添加%JAVA_HOME%\bin
  • classpath环境变量已在1.5版本弃用，这里不再赘述
  • 计算机”->右键 –>属性 –>进入”高级系统设置” 窗口—>进入”环境变量”窗口

  • 在“系统环境变量”窗口，点击“新建”–>打开“新建系统环境变量”窗口
  • 变量名为”JAVA_HOME”,变量值为jdk安装目录，如”D:\Program Files\Java\jdk”,点击”确定”
  • 然后找到path变量，点击”编辑”,加上”%JAVA_HOME%\bin”,点击”确定”

    验证环境变量是否配置成功

    win+R打开运行,输入CMD打打命令提示符窗口
    输入java,javac或java -version指令查看是否可看到JDK的编译器信息；

    验证是否正常工作

  • 在磁盘上创建一个Helloworld文件夹，用记事本编写一个“Helloworld”小程序来验证，保存为Helloworld.java，
  • 先输入javac Helloworld.java进行编译，得到Helloworld.class,再输入java Helloworld运行。
  • 安装结束
