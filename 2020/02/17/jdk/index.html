JDK1.8安装教程

发表于 2020-02-17

1.JDK的下载与安装

  • jdk8 x64下载地址
    链接
    提取码: 25th

  • jdk的安装
    下载完成后双击jdk-8u161-windows-x64.exe，点击“下一步”

    点击”更改”，选择要安装的目录

    更改完成后点击“下一步”

    正在安装

  • 安装完成后选择jre的安装目录，点击“下一步”

  • 安装完成，点击“关闭”

2.JDK环境变量配置

  • 新建 JAVA_HOME 环境变量,JAVA_HOME=C:\Java\jdk1.8.0_144

  • 修改 path 环境变量，在path变量尾部添加%JAVA_HOME%\bin

  • 新建 classpath 环境变量,classpath=.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\tools.jar
    （说明：jdk1.5之前需要配置3个环境变量，jdk1.5之后可以不用配置classpath）

  • 计算机”->右键 –>属性 –>进入”高级系统设置” 窗口—>进入”环境变量”窗口

  • 在“系统环境变量”窗口，点击“新建”–>打开“新建系统环境变量”窗口

  • 变量名为”JAVA_HOME”,变量值为jdk安装目录，如“D:\Program Files\Java\jdk”，点击“确定”

  • 然后找到path变量，点击“编辑”，加上“%JAVA_HOME%\bin”,点击“确定”

  • 新建 classpath 变量，设置 classpath=.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\tools.jar 点击“确定”，完成环境变量设置

    3.验证环境变量是否配置成功

    win+R打开运行，输入CMD打打命令提示符窗口
    分别输入java，javac，java -version指令查看是否可看到JDK的编译器信息；

    4.验证是否正常工作

    在磁盘上创建一个Helloworld文件夹，用记事本编写一个“Helloworld”小程序来验证，保存为Helloworld.java，

　　public class Helloworld{

　　　　public static void main(String[] args) {

　　　　System.out.println(“Hello Helloworld”);
　　　　}
　　}

　　先输入javac Helloworld.java进行编译，得到Helloworld.class,再输入java Helloworld运行。

  • 安装结束
