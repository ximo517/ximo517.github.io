MySQL解压版安装教程

发表于 2020-02-17

1.MySQL5下载地址：https://dev.mysql.com/downloads/mysql/5.7.html#downloads
选择要安装的版本进行下载
2下载完成后将文件解压，放到要安装的目录
说明：解压版没有data目录和ini文件
3.在D:\Program Files\MySQL\mysql-5.7.29目录下新建my.ini文件，复制如下内容
[mysqld]

port = 3306

basedir=D:\Program Files\MySQL\mysql-5.7.29

datadir=D:\Program Files\MySQL\mysql-5.7.29\data

max_connections=200

character-set-server=utf8

default-storage-engine=INNODB

sql_mode=NO_ENGINE_SUBSTITUTION,STRICT_TRANS_TABLES

[mysql]

default-character-set=utf8

说明：“D:\Program Files\MySQL\mysql-5.7.29”是我的MySQL安装目录
4.环境变量配置
此电脑 –> 右键 –> “属性” –>点击”高级系统设置”

–> 点击”环境变量”

在系统变量部分新建一个变量名：MYSQL_HOME， 变量值：D:\Program Files\MySQL\mysql-5.7.29

5 MySQL的安装
管理员身份运行cmd

进入MySQL安装目录，D:\Program Files\MySQL\mysql-5.7.29\bin

a 运行命令：mysqld –initialize (此时会生成data目录)

说明：
如果运行命令提示：由于找不到MSVCR120.dll,无法继续执行代码.重新安装程序可能…

这种情况需要安装 vcredist

下载vcredist ：https://www.microsoft.com/zh-CN/download/details.aspx?id=40784

下载后，直接安装。

b 运行mysqld -install （安装）
c 运行net start mysql (启动mysql服务)

可以在任务管理器中看到MySQL服务

6.设置root账户密码：

a 在my.ini文件（MySQL的配置文件）的[mysqld]下加一行skip-grant-tables

然后在任务管理器中重启MySQL服务

b重启MqSQL服务后，运行mysql -uroot -p,可以成功登入mysql

然后更新root账户的密码为’root’

命令：update mysql.user set authentication_string=password(“root”) where user=”root”;

然后输入flush privileges;（刷新账户信息）

执行quit或ctrl+Z退出

然后将my.ini文件中刚才加的skip-grant-tables这一行删掉，保存后再重启MySQL服务

c 然后运行mysql -uroot -proot就可以用root用户名和root密码登陆了

说明:
如果提示：You must reset your password using ALTER USER statement before executing this statement.

运行这条语句进行修改密码：alter user user() identified by “root”;
